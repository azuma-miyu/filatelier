# コンポーネント仕様書

Figmaで再現するためのコンポーネント仕様です。

## 📦 Header コンポーネント

### 仕様
- **種類**: AppBar（固定ヘッダー）
- **高さ**: 64px
- **背景色**: Primary Main (`#1976d2`)
- **テキスト色**: 白 (`#ffffff`)

### レイアウト
```
[Fil Atelier] [管理画面ボタン] [カートアイコン] [ユーザーアイコン]
```

### 要素
1. **ロゴ/タイトル**: "Fil Atelier" - H6タイポグラフィ
2. **管理画面ボタン**: 管理者のみ表示、アイコン + テキスト
3. **カートアイコン**: バッジ付き（商品数表示）
4. **ユーザーアイコン**: ドロップダウンメニュー

### スペーシング
- パディング: 16px（左右）
- 要素間のギャップ: 8px

---

## 🎴 ProductCard コンポーネント

### 仕様
- **幅**: グリッドベース（1カラム: 100%, 2カラム: 50%, 3カラム: 33.33%）
- **高さ**: 自動（コンテンツに応じて）
- **背景色**: 白 (`#ffffff`)
- **影**: Elevation 1

### レイアウト
```
┌─────────────────┐
│   [商品画像]     │  200px高さ
│                 │
├─────────────────┤
│ 商品名 (H6)      │
│ 説明文 (Body2)   │
│ ¥2,800  在庫:8  │
│ [カートに追加]   │
└─────────────────┘
```

### 要素
1. **商品画像**: 200px高さ、アスペクト比維持
2. **商品名**: H6タイポグラフィ、1行表示（省略可能）
3. **説明文**: Body2、100文字まで表示（省略記号付き）
4. **価格**: Primaryカラー、H6タイポグラフィ
5. **在庫表示**: Success（在庫あり）またはError（在庫切れ）
6. **カートに追加ボタン**: Primary、フル width

### スペーシング
- パディング: 16px
- 要素間のギャップ: 16px
- 画像とコンテンツ間: 0px（画像がカードの上部）

---

## 📋 ProductList コンポーネント

### 仕様
- **レイアウト**: CSS Grid
- **カラム数**: 
  - XS (モバイル): 1カラム
  - SM (タブレット): 2カラム
  - MD+ (デスクトップ): 3カラム
- **ギャップ**: 24px

### レスポンシブ
```css
grid-template-columns:
  xs: 1fr
  sm: repeat(2, 1fr)
  md: repeat(3, 1fr)
```

## 🔐 LoginPage コンポーネント

### 仕様
- **幅**: 最大 600px（sm）
- **中央揃え**: ページ中央
- **背景**: 白（Card内）

### レイアウト
```
┌─────────────────────┐
│     Fil Atelier     │  H4、中央揃え
│  [ログイン][新規登録] │  Tabs
│                     │
│  メールアドレス      │
│  ┌───────────────┐  │
│  │               │  │  56px高さ
│  └───────────────┘  │
│                     │
│  パスワード         │
│  ┌───────────────┐  │
│  │               │  │  56px高さ
│  └───────────────┘  │
│                     │
│  [ログインボタン]    │  48px高さ、フル width
└─────────────────────┘
```

### 要素
1. **タイトル**: "Fil Atelier" - H4、中央揃え
2. **タブ**: ログイン/新規登録切り替え
3. **入力フィールド**: 
   - 高さ: 56px
   - Border Radius: 4px
   - ラベル付き
4. **ボタン**: Primary、フル width、Largeサイズ

### スペーシング
- カードパディング: 32px
- フィールド間: 16px
- ボタンの上マージン: 24px

## 🛒 CartItem コンポーネント

### 仕様
- **レイアウト**: 横並び（Flexbox）
- **背景**: 白
- **影**: Elevation 1

### レイアウト
```
┌─────────────────────────────────┐
│ [画像] 商品名      ¥2,800  [×]  │
│ 64×64 数量: [−] 2 [+]          │
└─────────────────────────────────┘
```

### 要素
1. **商品画像**: 64×64px、角丸
2. **商品名**: Body1
3. **価格**: Body1、右揃え
4. **数量コントロール**: インクリメント/デクリメントボタン
5. **削除ボタン**: アイコン

---

## 📐 共通スタイル

### ボタン
- **Primary Button**: 
  - 背景: Primary Main
  - テキスト: 白
  - Border Radius: 4px
  - Height: 40px (Medium)

- **Text Button**:
  - 背景: 透明
  - テキスト: Primary Main
  - Border Radius: 4px

### カード
- **Elevation**: 1（軽い影）
- **Border Radius**: 4px
- **Padding**: 16px

### 入力フィールド
- **Height**: 56px
- **Border**: 1px solid rgba(0,0,0,0.23)
- **Border Radius**: 4px
- **Focus Border**: 2px solid Primary Main

### アイコン
- **サイズ**: 24×24px
- **色**: 親要素のテキスト色に従う

---

## 🎨 状態

### ボタン状態
- **Default**: Primary Main背景
- **Hover**: Primary Light背景（暗くする）
- **Disabled**: グレーアウト、クリック不可
- **Active**: Primary Dark背景

### 入力フィールド状態
- **Default**: グレーボーダー
- **Focus**: Primary Mainボーダー、2px
- **Error**: Errorカラーボーダー
- **Disabled**: グレーアウト
